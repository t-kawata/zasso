# Release v0.24.259

## Summary

Initial scaffolding of the `voiput` portable speech-to-text crate, extracted from the MYCUTE codebase. First 3 tickets (M0-1, M0-2, M1-1) implemented with full test coverage and a live demo binary.

## What's Changed

- **feat**: Created `voiput` crate skeleton with Cargo.toml, build.rs, lib.rs, error types, and STT constants extracted from MYCUTE
- **feat**: Added public type definitions (SttEvent, SttEngine, LocaleCode, VoiceKitConfig with builder pattern, and 5 config structs) with 28 tests
- **feat**: Ported SincResampler from MYCUTE (rubato-based high-quality sinc interpolation) with 5 tests
- **feat**: Added `test-run.rs` development demo binary (Stage 3/6) with CONFIG and RESAMPLER sections
- **docs**: Unified crate name from `voice_kit` to `voiput` across RFC document
- **chore**: Ticket management system with 3 completed/reviewed tickets
