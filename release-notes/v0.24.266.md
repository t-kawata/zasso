# Release v0.24.266

## Summary

Added `PseudoAsrStreamer` pipeline orchestrator, native FFI bindings for macOS/Windows, and OpenAI Whisper API backend. The voiput crate now has a functional speech-to-text pipeline with multiple backends.

## What's Changed

- **feat(voiput)**: Added `PseudoAsrStreamer` (M3-1) — VAD → Denoiser → SignalFilter → ASR → Punctuation → PostCorrection pipeline orchestrator
- **feat(voiput)**: Added `OpenAIBackend` (M4-2) — async-openai v0.41.0 integration with `call_transcribe()` (WAV → Whisper API) and `call_post_correct()` (LLM correction). Replaced MYCUTE's LmgwClient with `OpenAiConfig` + direct `async-openai::Client`
- **feat(voiput)**: Added `test_openai()` to test-run.rs — loads `sample-voice.wav`, creates Tokio runtime, and calls actual `transcribe()`. Default `base_url` is `https://api.openai.com/v1`
- **feat(voiput)**: Added native FFI bindings (M4-1) — `native/mac_ffi.rs` and `native/win_ffi.rs` with extern "C" declarations
- **feat(Makefile)**: Added branch operations — `make branch`, `commit-branch`, `push-branch` for non-master workflows
- **chore**: 3 tickets completed and reviewed (M3-1, M4-1, M4-2)
