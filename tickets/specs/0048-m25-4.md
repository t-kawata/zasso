# M2.5-4: 移行後の動作確認

## Summary

M2.5-1 〜 M2.5-3 で実施した sherpa-rs → sherpa-onnx 移行が完全であることを確認する。
`cargo test` 全件 PASS、`cargo run --bin test-run` 正常動作、Cargo.toml ・ build.rs のクリーンアップを確認する。

## Background

M2.5-1 〜 M2.5-3 を連続実行し、ビルドは回復済み。本チケットでは移行の完全性を確認する。

## Scope

### 確認項目

1. **`cargo test` 全72テスト PASS**（既に PASS 確認済み）
2. **`cargo run --bin test-run` 正常動作**（既に確認済み）

4. **`pipeline/denoiser.rs` に sherpa_rs_sys 参照がないこと**
