## Background

`docs/rfc-stt-portable-crate.md` と実装の間に乖離が生じている。実装が進化したが RFC が追従していない項目（5件）、実装が RFC より改善されている項目（3件）、軽微な差異（5件）の計13項目を修正する。

## Investigation

### 🔵 RFC未更新（RFCに実装を反映）

| # | 項目 | RFC の記述 | 実際の実装 |
|---|------|-----------|-----------|
| 8 | build.rs (§9) | 単純なプリビルド存在確認＋panic (50行) | 自動ビルド・スタブ生成・libs/収集・Swift Concurrency チェック等 (816行) |
| 9 | Denoiser (§7.6) | `sherpa_rs_sys` unsafe FFI | `sherpa_onnx` safe API |
| 10 | Cargo.toml (§8) | `sherpa-rs = "0.6"`, `sherpa-rs-sys = "0.6"` | `sherpa-onnx = "1.13.2"` |
| 11 | macOS build.sh (§6.1) | ユニバーサルバイナリ, `-target`, `-O -whole-module-optimization` | arm64のみ、フラグなし |
| 12 | Windows build.ps1 (§6.2) | `PublishAot=true`, `NativeLib=Shared`, 特定コピー元 | 簡略化された引数 |

## Scope
